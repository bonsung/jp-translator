# JP 번역 도구 (JP Translator)

화학 무역 실무에 특화된 일본어 ↔ 한국어 번역 도구입니다.

## 주요 기능

- **다중 번역 모드**: 일반 번역 / 화학 무역 전문 번역 / 이메일 작성 / 견적 해석
- **단일 HTML 파일**: 별도 설치 없이 브라우저에서 바로 실행
- **모바일 지원**: 반응형 디자인으로 스마트폰에서도 사용 가능
- **Claude AI 연동**: Anthropic Claude API를 사용한 고품질 번역

## 사용법

1. [배포 주소]에서 바로 사용하거나
2. `jp-translator.html` 파일을 다운로드 후 브라우저로 열기
3. Anthropic API Key 입력 (최초 1회)
4. 번역 모드 선택 후 텍스트 입력 → 번역 실행

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- Anthropic Claude API (claude-sonnet-4-6)
- 단일 파일 구조 (외부 의존성 없음)

## 대상 사용자

중국 → 한국 화학제품 무역 실무자 (MEK, NBA, 2-EH, EA, Propylene 등)
